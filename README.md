# BrightSmile Dental Care

A premium, conversion-focused landing page concept for a modern dental clinic in
Chennai, India — *"Advanced Dental Care for a Healthier Smile."*

> **Portfolio demo:** This is a concept website created to showcase web design and
> development skills. "BrightSmile Dental Care" is a fictional brand, not a real clinic.

## ✨ Highlights

- **Premium healthcare design** — clean white & blue palette, trustworthy and modern.
- **Mobile-first & responsive** — sticky mobile CTA bar (Call · Book · WhatsApp),
  floating WhatsApp button, and click-to-call throughout.
- **Lead generation** — appointment booking form, appointment popup modal,
  **exit-intent popup**, and qualified capture (treatment + preferred date).
- **SEO optimized** — title/meta/Open Graph tags, **Dentist/LocalBusiness schema**,
  **FAQ schema**, Chennai-focused keywords and semantic markup.
- **Performance** — self-hosted WebP images + woff2 fonts, preloaded critical assets,
  lazy-loading and async decoding.

## 🗂 Structure

```
index.html          # markup + JSON-LD schema
styles.css          # styling
fonts.css           # self-hosted @font-face
script.js           # nav, sliders, popup, exit-intent, forms
imges/              # optimized WebP images
fonts/              # self-hosted woff2 (Poppins + Inter)
optimize_images.py  # asset pipeline
optimize_fonts.py   # asset pipeline
```

## 🚀 Run locally

```bash
npx serve .
```

Hand-written HTML, CSS and vanilla JavaScript — no frameworks or build step.
